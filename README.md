# Sentiment Analysis Dashboard

A web-based dashboard to analyze customer reviews using NLP.

#Live Demo
https://sentiment-analysis-sodm.onrender.com/

#Features
- Sentiment classification (Positive, Negative, Neutral)
- Multiple review input
- Data visualization (charts)
- SQLite database with CRUD operations

#Tech Used
- Python
- Flask
- TextBlob
- SQLite
- Matplotlib
